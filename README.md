# 📌 Overview

InvoiceIQ is an advanced AI-powered fintech platform built for the **CodeFlow 2026 Hackathon** under the theme:

> **Artificial Intelligence for FinTech**

The application allows users to upload invoices, bills, or receipts in image/PDF format and automatically extracts, categorizes, analyzes, and generates intelligent financial insights using AI and Machine Learning.

Unlike traditional OCR-based systems, InvoiceIQ acts as a complete **AI Financial Intelligence Engine** capable of:

- Understanding invoice data
- Tracking spending behavior
- Detecting fraud & duplicate invoices
- Predicting future expenses
- Generating AI-driven financial summaries
- Providing smart budgeting recommendations

# ⚙️ Installation Guide

---

# 1️⃣ Clone Repository

```bash
git clone https://github.com/AyanAlikhan11/CODEFLOW2026-TEAMQUORIX-InvoiceIQ.git

cd CODEFLOW2026-TEAMQUORIX-InvoiceIQ
```

---

# 2️⃣ Install Frontend Dependencies

```bash
npm install
```

---

# 3️⃣ Run Frontend

```bash
npm run dev
```

Frontend runs on:

```bash
http://localhost:3000
```

---

# 🐍 AI Service Setup

---

# 4️⃣ Navigate to AI Service

```bash
cd ai-service
```

---

# 5️⃣ Create Virtual Environment

## Windows

```bash
python -m venv venv
venv\Scripts\activate
```

## Linux / Mac

```bash
python3 -m venv venv
source venv/bin/activate
```

---

# 6️⃣ Install Python Dependencies

```bash
pip install -r requirements.txt
```

---

# 7️⃣ Run FastAPI Server

```bash
uvicorn main:app --reload
```

AI service runs on:

```bash
http://localhost:8000
```

---

# 🔑 Environment Variables

Create a `.env` file in the root directory.

```env
NEXT_PUBLIC_API_URL=http://localhost:3000

AI_SERVICE_URL=http://localhost:8000

DATABASE_URL=your_database_url

OPENAI_API_KEY=your_openai_key

GEMINI_API_KEY=your_gemini_key
```
